
* To push to Amazon S3 bucket
* will need to install s3_website plugin
* type 'gem install s3_website'
* then create cfg file, 's3_website cfg create'
* then apply cfg file,  '3_website cfg apply'
* to push the build jekyll _site, type 's3_website push'
* here is the doc, https://github.com/laurilehmijoki/s3_website

S3
s3_website cfg create
3_website cfg apply
s3_website push
